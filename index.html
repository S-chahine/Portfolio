Quote Machine

The Quote Machine App is a web application built with React, Bootstrap, and FontAwesome. It dynamically fetches random quotes from the Quotable API, presenting them in an aesthetically pleasing card layout. The app features a vibrant color scheme that changes with each new quote, enhancing the visual experience. Users can easily generate a new quote, tweet their favorite quotes, and share them on Instagram. The responsive design ensures an optimal viewing experience across various devices.

Key Features:

  • Random quote generation from the Quotable API
  • Dynamic color scheme for a visually engaging experience
  • Tweet quotes directly from the app
  • Share quotes on Instagram with a single click

Technologies Used:

  • React
  • Bootstrap
  • FontAwesome
  • Quotable API
